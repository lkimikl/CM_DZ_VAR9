# Конфигурационный преобразователь (Вариант №9)

Инструмент командной строки для преобразования учебного конфигурационного языка в YAML.

## Описание

Данный инструмент преобразует текстовые файлы, написанные на учебном конфигурационном языке (вариант №9), в формат YAML. Поддерживаются все конструкции языка, включая константные выражения и функции.

## Синтаксис учебного конфигурационного языка

### Комментарии
- Однострочные комментарии: `// комментарий`

### Числа
- Формат: `[+-]?\d+` (целые числа со знаком)

### Массивы
- Формат: `array(значение, значение, значение, ...)`

### Строки
- Формат: `"строка"`

### Имена
- Формат: `[a-zA-Z][_a-zA-Z0-9]*`

### Объявление констант
- Формат: `global имя = значение`

### Константные выражения (префиксная форма)
- Формат: `@[операция аргументы]`
- Пример: `@[+ имя 1]`

### Поддерживаемые операции и функции
- **Арифметические операции**: `+`, `-`, `*`, `/`
- **Функции**: 
  - `ord()` - возвращает код символа
  - `abs()` - возвращает абсолютное значение
